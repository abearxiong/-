使用下方代码获取总的字数数量。
```
{% raw %}
{{ page.content | size }}
{% endraw %}
```

会获取包括编程`{  }`这些标志的数量，是所有内容数量的总和。

一个汉字，字母，字符都是1，entrl也是1

字数： {{ page.content | size }}

注意到，如果文章中的内容超过某一个限度的时候，是不能解析的。
