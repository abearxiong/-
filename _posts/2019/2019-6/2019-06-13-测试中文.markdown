## markdown中文名字

文件的名字可以是中文的。

-[x] 关内容一定要有description,行数一定不能超过三行，一定是文章的总结。之前的文件，一定要有描述，而且描述一定在三行一下。规范化页面显示效果。
-[x] 关于ruby中创建新的内容，把设置文件的部分和标题这部分，重构掉，因为中文名是可以的，中文的url地址也是可以访问的
-[x] 把首页的内容，重构。标题 描述 标签 更新时间
-[x] github issues 是草稿纸或者就是属于空间。 随心play..
