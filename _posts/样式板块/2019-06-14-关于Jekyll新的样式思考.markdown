## 关于style内容

无论是html亦或者md，markdown文档，都可以在内容中添加标签，添加style，添加脚本sript，然后通过这些都可以执行那些样式内容。

比如下面的例子
```
{% raw %}
<style type="text/css">
    .right{
        position: relative;
        top: 20px;
        width: 100%;
        height: 20px;
        color: red;
    }
</style>
<div class="right">查看一下内容会往左边靠吗？？？</div>
{% endraw %}
```
